Типи CSS Селекторів

  • Універсальні селектори CSS позначаються звичайною зірочкою "*". Під зірочкою може матися на увазі будь-який елемент розмітки HTML-сторінки.
  • Селектори за назвою тега. Як селектор виступає ім'я тега, для якого необхідно змінити властивості.
  • Селектори за класами. "class" - це атрибут HTML-тега, може застосовуватися до однотипних елементів
  • Селектори по id. Цей атрибут застосовується до індивідуального елементу, тобто. елемент з id може бути в єдиному екземплярі на сторінці згідно з правилами семантики.
  • Селектори за атрибутами. Дозволяють стилізувати елемент як за значенням тега, а й за значенням атрибута
  • Селектори нащадків, або контекстні селектори.мається на увазі ситуація, коли один елемент HTML-сторінки вкладений в інший, причому не обов'язково він повинен бути першим нащадком.
  • Дочірні селектори. Дочірні селектори CSS відрізняються від селекторів нащадків тим, що під дочірнім мається на увазі лише перший нащадок, ніякі "онуки" та "правнуки" не потрапляють до цієї категорії.
  • Сестринські, або сусідні селектори. За допомогою сестринського селектора створюється CSS-правило для HTML-елементів, розташованих на одному рівні, причому той елемент, який записується як сестринський, повинен бути найближчим із себе подібних до того елемента, до якого він записується в "сестри".
  • Селектори псевдокласів. До селекторів псевдокласів відносяться селектори CSS, які описують стан елементів при здійсненні над ними будь-якої дії. Це може бути наведення елемент курсору миші, клік по елементу тощо. Записуються вони через двокрапку до елемента, до якого застосовуються.
  • Селектори псевдоелементів. За їх допомогою (::before, ::after) можна додавати різні елементи на веб-сторінки, але при цьому псевдоелементи ніяк не будуть відображені в HTML-розмітці цих сторінок. Записуються вони по відношенню до елемента через подвійну двокрапку.
  • Селектори за вкладеністю. Дозволяють вибирати елементи, які знаходяться всередині певних інших елементів в ієрархії HTML.
  • Селектори за кількома класами. Дозволяють вибирати елементи, які мають одночасно декілька класів
